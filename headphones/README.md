Headphone Company – Landing Page

This project is a responsive landing page built from the provided Figma design. The goal is to accurately recreate the layout, typography, spacing, and interactions while maintaining clean and organized code.

 Features

Fully responsive layout (Desktop + Mobile at ≤480px).

Pixel-accurate structure based on the Figma file.

Hover/active interactions:

Links hover/active color: #FF5655

Button hover/active: opacity: 0.9

Max content width: 1000px, centered.

Uses the required fonts:

Source Sans Pro

Spin-Cycle-OT


 How to Run

Open the project folder.

Open index.html in any browser.

Ensure fonts are installed or linked properly for correct rendering.

Design Source

The original layout comes from the provided Figma file.
I duplicated it to my drafts and used it as the reference for spacing, colors, typography, and layout.

 Responsive Behaviour

When the screen width is 480px or below, the layout switches to mobile view.

Content stacks vertically and spacing adapts for smaller screens.
